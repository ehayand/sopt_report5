# Pokemon Book URI



| 메소드 | 경로                   | 설명             |
| ------ | ---------------------- | ---------------- |
| GET    | /pokemons              | 모든 포켓몬 조회 |
| GET    | /pokemons/{pokemon_id} | 포켓몬 조회      |
| POST   | /pokemons              | 포켓몬 등록      |
| PUT    | /pokemons/{pokemon_id} | 포켓몬 수정      |
| DELETE | /pokemons/{pokemon_id} | 포켓몬 삭제      |
